# json2ChinaRegions


## 生成省市区JSON文件

目前只有一种格式

```
      
[
    {
        "name":"省名",
        "city":[
            {
                "name":"城市名",
                "area":[
                    "区名",
                    "区民"
                ]
            }
        ]
    },
    {
        "name":"省名",
        "city":[
            {
                "name":"城市名",
                "area":[
                    "区名",
                    "区民"
                ]
            }
        ]
    },
    {
        "name":"省名",
        "city":[
            {
                "name":"城市名",
                "area":[
                    "区名",
                    "区民"
                ]
            }
        ]
    }
]
```

## 数据来源
最全最新中国省，市，地区 json，来自国标 http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/ 数据。
